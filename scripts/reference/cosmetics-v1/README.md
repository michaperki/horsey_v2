# Cosmetics v1 — archived assets

The avatar art from the ripped v1 cosmetic system (removed 2026-05-25).

- `assets/manifest.json` — the v1 catalog: anchors, z-order, density modes, per-item asset metadata.
- `assets/*.png` — base piece art (knight/queen/pawn/bishop/rook), trust-border rings
  (provisional/verified/trusted/elite/champion), the veteran badge.
- `assets/*.Zone.Identifier` — WSL/Windows download metadata; safe to ignore.

These files are kept as visual reference for the next cosmetic system pass. They
are not loaded by the running app. See `docs/archive/COSMETICS_INVENTORY_AUDIT.md`
for the audit that classified each asset.
